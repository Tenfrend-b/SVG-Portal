# SVG-link-for-MT-manager

当你使用MT管理器时，是否有想过像使用Windows时那样自定义快捷方式（MT里就是书签）的的图标？或者是做一个快捷方式大合集，把一大堆指向各种位置的快捷方式分门别类的放好？

MT管理器原版在这方面的处理简洁优雅，但也使得类似上述功能存在缺失。因此，在深度使用MT管理器很长时间后，一个实现类似功能的思路慢慢串成一线，并最终在Deepseek “V3最终版”的帮助下实现了这个软件。

使用教程(●°u°●)​ 」
1. 安装本软件。由于本软件没有启动类，因此不会在安装后提示“打开”选项或在桌面生成图标，属于正常现象

2. 找一个好看的SVG文件，在文件的开头添加一行注释，内容就是要指向的目录。大抵就是在文件开头添加&lt;!--/path/to/your/dir--&gt;这样的东西

3. 在MT管理器中长按文件，将本软件设置为SVG文件的默认打开方式。第一次打开后就会提示设置所有文件管理权限（或者是储存权限），照做即可

4. 再次点击SVG文件，你就能发现MT目录神奇的跳转啦（｡ò ∀ ó｡）

实现过程：
事实1: MT管理器将自身注册为一种文件打开方式，其打开行为就是直接定位到该文件所在的位置

事实2: MT管理器可设置某种文件的默认打开方式

事实3: 不管SVG文件的默认打开方式为何，MT都会默认渲染出其预览图

事实4: SVG就是XML，本身就是便于编辑的文本文件，且支持注释

因此获得以下思路：

MT设置本软件为SVG的默认打开方式→本软件获得SVG正文后按特定方法解析出目标路径→本软件请求MT打开目标路径的文件→MT成功打开，目录成功跳转→实现win风格快捷方式功能


软件注意事项：
1. 目前此软件对快捷方式文件的位置没有限制，但对跳转到的目录有限制。MT管理器与此软件必须同时具有对目标位置的访问权限（换句话说就是/sdcard/Android/data与/data/data/xxx不可以）
2. 如果需要卸载软件但由于软件没有桌面图标不知如何操作，可在MT管理器的安装包提取功能页面删除软件，或者你有其他办法也行。
3. 本软件代码100% AI生成，所以如果发现了什么bug欢迎报上来哈
